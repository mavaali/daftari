# Corpus (B) CB6 — The keystone, measured: tension-masquerade on "no consensus" items

**Date:** 2026-06-29
**Design:** `docs/plans/2026-06-29-two-corpus-sovereignty-paper-design.md` (CB6 section).
**Models:** foil panel = `anthropic/claude-haiku-4.5`, `z-ai/glm-4.6`, `openai/gpt-4o`;
daftari detector = Haiku (Anthropic, faithful to daftari's client); second-rater =
`google/gemini-2.5-flash` (cross-family, independent of all foils). One run, ~$1.

**Verdict:** the keystone — *"a tension may never masquerade as a supersession"* — moves
from *by-construction* to **measured**. A consolidation memory that must emit a single
current value masquerades a genuine tension as a supersession **4/4, identically across
all three models** (forced condition); daftari mints **0** and manufactures **0** false
conflicts — it preserves both positions. The honest softness: given an explicit abstain
option, a *capable* model abstains most (GPT-4o 1/4) while conservative models still
masquerade (Haiku/GLM 3/4) — so daftari's guarantee is *architectural*, the LLM's is
*contingent on model + an escape hatch*.

## Setup

The consensus box's **"no consensus" items are genuine tensions**: the status quo holds
*by default*, not by superseding the alternative on the merits — editor-labeled (e.g. #48
*"…there is no consensus on specific wording, but the status quo is X"*). Four are
currently active (Trump box): **#15** (2016 election phrasing), **#48** (COVID-response
wording), **#56** (Russian-bounties wording), **#65** (Abraham Accords framing). For each,
the two competing positions were distilled from the linked RfC into a pair
`(statusQuo, alternative)` with **ground truth = NEITHER supersedes**, then **gated by a
blind cross-family second-rater** ("is this a genuine unresolved disagreement where
neither has won out?") — **4/4 validated** (the pairs are not strawmen). No LLM labeler
for the ground truth (the editor "no consensus" label is the truth) → no contamination.

Two foil conditions per model:
- **Forced** (no NEITHER option) — models a consolidation memory whose architecture has no
  tension slot; it *must* pick a direction.
- **Abstain-offered** (the CB4 foil, which offers NEITHER) — models an LLM consolidation
  step that *could* decline. daftari's path = the CB5 contradiction detector (never asks
  direction → mints 0).

## Reading

- **The keystone, as an architectural fact (forced, 4/4, model-independent).** A memory
  that must store one current value cannot represent a tension — so it collapses every one
  into a supersession, and this does **not** depend on model capability (GPT-4o masquerades
  4/4 just as Haiku does, when it cannot abstain). This is the precise threat daftari's
  never-mint guarantee answers. The forced verdicts are also **position-biased** (mostly
  `b_supersedes_a`, the second-presented), same as CB4 — the minter isn't reasoning about
  supersession, it's manufacturing one.
- **daftari preserves both — minting 0 *and* asserting 0 false conflicts.** The detector
  flagged only **#65** (significant vs ceremonial — a genuine contradiction); it correctly
  said NO on #15/#48/#56, which are **framing/detail disputes, not logical contradictions**
  ("2.5M more votes" and "a larger popular-vote share" are both true). This is *not* a
  daftari miss — it is preserve-not-resolve in its purest form: daftari neither mints a
  supersession nor manufactures a contradiction where there is only a framing dispute. It
  holds both positions for human resolution (`vault_tension_log`). Auto-detection of these
  framing-tensions is low **by the same design** as CB5; the guarantee daftari makes is
  **no-mint**, not auto-acquisition.
- **The honest softness (abstain-offered).** With an explicit escape hatch, masquerade is
  model-dependent — GPT-4o abstains on 3/4. So "auto-minting fabricates" is true of the
  *architecture* (forced) and of *conservative* models (Haiku/GLM 3/4), but a careful,
  capable, abstain-prompted LLM approximates daftari's restraint. daftari's edge is that
  the restraint is **guaranteed by construction**, not contingent on picking the right
  model and offering the right option.

## Honest precision

- **n = 4** (the Trump box's active "no consensus" items). Small. Scale = pull additional
  articles' boxes (Biden, Reagan, COVID) for more "no consensus" items — the editor-label
  cleanliness generalizes.
- **Pairs are distilled, then second-rater-gated.** The status-quo side is verbatim from
  the box; the alternative is distilled from the RfC thread (a judgment step), so the
  second-rater gate (4/4) is load-bearing, not decorative.
- **Single run; LLM nondeterminism is visible** (GLM's abstain-offered rate moved 2/4→3/4
  across two runs at temp 0). The forced 4/4 is stable; the abstain-offered numbers should
  be read as one sample, not a point estimate.
- **The forced condition is near-tautological by design** — that is the point (it models an
  architecture, not a behavior); it is reported *with* the abstain-offered condition so it
  is contextualized, not the only number (per the design's failure-mode check).

## For the paper

CB6 is the keystone section of the two-corpus paper: the forced 4/4 (model-independent) is
the architectural statement; the abstain-offered range is the honest softness; daftari's
0-mint / 0-false-conflict is the preserve-not-resolve demonstration. Combined with the
structural guarantee (the loop cannot mint), the spine is no longer by-construction alone.
